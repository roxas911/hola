Introduccion a Git

Comandos Basicos

  • git init: Inicializa un repositorio git
  • git status: muestra el estado de los archivos del proyecto
  • git add: Agrega archivos al area de preparado
  • git commit -m "mensaje":Guarda los archivos preparados en el repositorio
  • git log: Muestra el historial de commits que se ha hecho
  • git diff:Muestra diferencias entre archivo de proyecto
  • git checkout:permite moverse entre el historial de versiones
  • Backtracking
  • git checkout HEAD filename:descarta cambios en el area de trabajo
  • git reset HEAD filename: descarta cambios en el area de preparado

    • Branching

  • git branch
  • git branch branch_name
  • git checkout branch_name
  • git merge branch_name
  • git branch -d branch_name
